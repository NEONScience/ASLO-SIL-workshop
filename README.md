# ASLO-SIL-workshop
Repo for ASLO-SIL data harmonization workshop

Re-use of publicly available data in aquatic science is becoming increasingly common. Pooling of data across distributed sources has numerous advantages including expanded geographic coverage and increased modeling power. FAIR (Findability, Accessibility, Interoperability and Reusability) have become guiding principles in scientific data management. Recent advances in AI have greatly reduced the major challenges associated with findability and accessibility, however extensive human-guided, time-consuming "data wrangling" is often required before data are truly “AI ready”. This workshop seeks to address these challenges of making distributed datasets truly interoperable and reusable.
